MEASUREMENTS:

				"V0 defines a new round of measurements, this implies that the instrument unknowns are recalculated.\n"
				"It accepts a redefinition of the default target ID and the constant angle:\n"
				"USAGE:\n"
				"    *V0 [TRGT] [ACST]\n"
				"----------\n"
				" TRGT<str>: updated default traget ID\n"
				" ACST<flt>: overrides constant angle value\n"
				
				
				"TSTN (Total Station) defines the measurements of a total station from a certain point.\n"
				"It is a single line command that accepts no further lines except the following keyword *V0.\n"
				"USAGE:\n"
				"    *TSTN A B [ ROT3D | [ IHFIX | IH [IHSE] ]] [TRGT] [ICSE]\n"
				"----------\n"
				" A: str:   Point ID on which the station is positioned\n"
				" B: str:   Instrument ID of the total station\n"
				" ROT3D:    This positioning introduces three unknown orientation parameters\n"
				" IHFIX:    The instrument height will be fixed\n"
				" IH<flt>   Redefine the default instrument height\n"
				" IHSE<flt> Define the sigma of the instrument height, optional to IH\n"
				" TRGT<str> Change the default target ID\n"
				" ICSE<flt> Redefine the default target centering sigma\n"
				" \n"
				" EXAMPLES:\n"
				" *TSTN	P1  Theo1\n"
				" *TSTN	P2  Theo1  IHFIX IH 0.19\n"
